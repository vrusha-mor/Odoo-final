# POS Payment System

A comprehensive Point of Sale (POS) payment system with UPI QR generation, QR scanning, card payments, and email receipts.

## Features

- POS UI for product selection and cart management
- UPI QR Code generation for payments
- QR Scanner for payment verification
- Card Payment Redirect
- Email Receipt functionality
- Backend with Node/Express and Nodemailer for email sending
- Payment status tracking

## Tech Stack

- Frontend: React with TypeScript, Vite
- Backend: Node.js, Express
- Email: Nodemailer
- QR Generation: qrcode library

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies for frontend and backend
3. Set up environment variables
4. Run the application

### Running the Application

- Frontend: `npm run dev`
- Backend: `npm start`
